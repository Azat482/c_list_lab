# Связный список на C

Простая программа на С, реализующая односвязный список с функциями добавления, вставки, удаления и вывода элементов.

## Функции:

- Добавить элемент
- Вставить элемент по индексу
- Удалить элемент по индексу
- Показать список

## Сборка и запуск:

```bash
gcc -o list_app main.c
./list_app
```
